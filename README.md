# Java IP Address Splitter

This is a simple Java console program that allows users to input an IP address, splits the address into its four parts (octets), and prints each part on a separate line.

## Features

- Reads an IP address from the user.
- Splits the IP address into octets using the `split()` method.
- Prints each octet on a new line.

## How to Use

1. Clone or download this repository.
2. Open the project in your favorite IDE (like Eclipse or IntelliJ) or use the command line.
3. Compile the Java file:
   ```bash
   javac Main.java
